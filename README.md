# Tailwind CSS Landing Page
This repository contains a Tailwind CSS Landing Page project that I built as part of a tutorial I was following to learn **Tailwind CSS**.
The main goal of this project was to apply what I learnt and gain hands-on experience with utility-first CSS styling.

## Features
- Responsive layout using Tailwind breakpoints
- Content sections styled with Tailwind utility classes

## Tech Stack
- HTML
- Tailwind CSS
